# Ariel M. Noja — Full-Stack Developer Portfolio

Welcome to the source code for my personal developer portfolio! This project showcases my skills, experience, and the projects I've built. 

## 🛠️ Built With
*   **HTML5**: Semantic and accessible structure.
*   **CSS3**: Custom styling with variables, glassmorphism, flexbox/grid, and smooth keyframe animations (No CSS frameworks used).
*   **Vanilla JavaScript**: Handling scroll reveals, typing effects, dynamic data fetching, and particle canvas animations.
*   **FontAwesome 7**: Used for scalable vector icons throughout the UI.
